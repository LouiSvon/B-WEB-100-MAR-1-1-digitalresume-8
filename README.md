# Digital Resume

## Description

Digital Resume est une page web statique presentant un CV personnel en ligne.

## Objectif

L'objectif est de construire une page de presentation professionnelle avec HTML et CSS, en structurant les sections principales d'un CV : presentation, competences, experience, formation et contact.

## Stack / technologies

- HTML
- CSS

## Structure

```text
.
└── day04/
    └── digital resume/
        ├── digitalresume.html
        └── digitalresume.css
```

## Fonctionnalites principales

- Header avec navigation interne.
- Section de presentation.
- Cartes pour les competences, experiences et formation.
- Formulaire de contact statique.
- Mise en page personnalisee en CSS.

## Lancement

Ouvrir le fichier HTML dans un navigateur :

```bash
open "day04/digital resume/digitalresume.html"
```

## Remarques

Certaines images referencees dans le HTML pointent vers des chemins locaux macOS. Elles peuvent ne pas s'afficher sur une autre machine sans adaptation.
